What I've Accomplished

This page highlights some of the projects I have completed as part of the University of Washington's certificate in front-end web development curriculum.

Lesson 01: Semantics Basics This link opens in a new tab.

Our first assignment was to create a simple web page describing the difference between semantic elements and those that are nondescript or imply no inherent meaning.

Lesson 04: Kitties to Go This link opens in a new tab.

For this assignment, we were tasked to recreate a website called Kitties to Go based on an initial mockup, keeping in mind best practices for responsive web design and maintaining appropriate semantic structure in our design.

Lesson 08: Straight to the Point Archery This link opens in a new tab.

This project involved developing an initial outline of a website, based on a set of assets provided by a mock client, and creating a fully fledged set of pages out of the design. This mock site had to include CSS Grid and Flexbox elements, as well as semantic HTML.
